# README.md for building executables from the C sources

### To install kissfft and cjson C libraries from repository
```
sudo apt install libkissfft-dev
sudo apt install libcjson-dev
```
### To install kissfft and cjson C libraries from source code
```
git clone https://github.com/mborgerding/kissfft.git
git clone https://github.com/DaveGamble/cJSON.git
```
